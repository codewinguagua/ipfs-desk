# playground for ipfs.js with cljs and om.next

## setting up ipfs

* install ipfs (https://ipfs.io/)
  * `ipfs init`
  * `ipfs daemon`

* cljs
  * start figwheel `lein run -m clojure.main --init scripts/figwheel.clj`


_Drag files into the app to add them_
